# A-SOUL Medle

[**medle.infedg.xyz**](https://medle.infedg.xyz/)

曲库为A-SOUL五位成员（向晚，贝拉，珈乐，嘉然，乃琳）原唱曲及翻唱曲的旋律猜谜游戏，每日更新~

A-SOUL melody puzzle game (available in en/zh)

## [zh] 服务端程序

安装 Node.js，然后运行：

```
npm install
npm run start
```
